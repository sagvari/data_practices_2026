5 Short Coding Assignments with babynames

Setup

library(dplyr)
library(babynames)

These exercises focus only on row manipulation with:

  • filter()
  • arrange()
  • slice_head() / slice_tail()
  • slice_min() / slice_max()
  • slice_sample()

Each exercise also asks you to briefly interpret your result. After running the code, write 1–2 sentences describing what you found and what it might mean socially.

Exercise 1 — A very simple filter

Create a table that contains only female babies named "Anna" in the year 2000.

Print the result.

Scientific question

How common was the name Anna among girls in 2000 in the United States?

Reporting task

Write 1–2 sentences describing:

  • how many babies were named Anna
  • what percentage (prop) of all girls received that name

What might this tell us about traditional vs. fashionable names?

Hint

Use filter() and check:

  • name == "Anna"
  • year == 2000
  • sex == "F"

Look at the variables n and prop.

Exercise 2 — Filter and sort

Find all rows for the name "Jacob" among male babies, then sort the results from the largest n to the smallest n.

Print the 10 most popular years for the name Jacob.

Scientific question

When was Jacob most popular as a boys' name?

Reporting task

Write 1–2 sentences describing:

  • the year when Jacob peaked
  • whether the name seems to have stable popularity or a short boom period

Why might some names suddenly become very popular?

Hint

First keep only rows where:

  • name == "Jacob"
  • sex == "M"

Then sort with:

arrange(desc(n))

Use slice_head(n = 10) to keep the first 10 rows.

Exercise 3 — Most popular female names

Look only at female names in 2015. From that subset, return the 5 most common names.

Important: use a slice_... function.

Scientific question

Which names dominated among girls in 2015?

Reporting task

Write 1–2 sentences describing:

  • the top name
  • whether the top five names together seem to represent a large or small share of births

What might this say about diversity of names in modern societies?

Hint

Start with:

filter(year == 2015, sex == "F")

Then use:

slice_max(order_by = n, n = 5)

Exercise 4 — Looking at the "tail" of the distribution

Create two small tables from the year 1980:

  1. the first 4 rows of the 1980 subset
  2. the last 4 rows of the 1980 subset

Name them something like first_1980 and last_1980.

Scientific question

What do the least frequent names in a year look like?

Reporting task

Write 1–2 sentences describing:

  • how rare the names in the tail are
  • what this tells us about the long tail of name diversity in society

Why might parents choose extremely rare names?

Hint

First create a filtered subset:

filter(year == 1980)

Then use:

slice_head(n = 4)

and

slice_tail(n = 4)

Exercise 5 — Rare names

Among male names in 1995, find the 10 least common names.

Return exactly 10 rows.

Scientific question

How rare can names be in a national dataset of births?

Reporting task

Write 1–2 sentences describing:

  • the value of n for the rarest names
  • whether the dataset contains many extremely rare names

Why might rare names be sociologically interesting?

Hint

Use:

filter(year == 1995, sex == "M")

Then:

slice_min(order_by = n, n = 10, with_ties = FALSE)

with_ties = FALSE ensures you get exactly 10 rows.

Optional Bonus — Random sampling

Take the subset of all names from 2020 and draw a random sample of 8 rows.

Scientific question

What does a random sample of names from one year look like?

Reporting task

Write 1–2 sentences describing:

  • what kinds of names appear in the random sample
  • whether they seem common or rare

What does this illustrate about sampling from large datasets?

Hint

Use:

set.seed(123)

Then:

slice_sample(n = 8)
