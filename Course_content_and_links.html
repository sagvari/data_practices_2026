How to install R and R studio?

Installing R and RStudio setup guide

You need to install:

  • R (the statistical computing engine)
  • RStudio Desktop (the user-friendly interface for R)

1️⃣ Install R (First!)

R must be installed before installing RStudio.

Official R download page:
https://cran.r-project.org/

Windows Installation

Step 1: Download R

  1. Go to: https://cran.r-project.org/
  2. Click “Download R for Windows”
  3. Click “base”
  4. Click “Download R-x.x.x for Windows” (latest version)
  5. Run the downloaded .exe file
  6. Accept default settings during installation

That’s it — R is now installed.

macOS Installation

Step 1: Download R

  1. Go to: https://cran.r-project.org/
  2. Click “Download R for macOS”
  3. Download the latest .pkg file
  4. Open the file and follow installation instructions
  5. Accept default settings

💡 On newer Macs (Apple Silicon – M1/M2/M3), download the version marked for Apple Silicon (arm64) if available.

2️⃣ Install RStudio Desktop

After R is installed, install RStudio.

Official RStudio Download Page:
https://posit.co/download/rstudio-desktop/ (RStudio is developed by Posit.)

Windows & macOS

  1. Visit the link above
  2. Click “Download RStudio Desktop”
  3. Choose the free RStudio Desktop (Open Source License)
  4. Download the installer for your system
  5. Run the installer
  6. Accept default settings

3️⃣ Test Your Installation

After installing:

  1. Open RStudio
  2. In the Console (bottom left panel), type:
2 + 2

If you see:

[1] 4

🎉 Congratulations! Your installation works.

4️⃣ Recommended Settings (Optional but Helpful)

In RStudio:

  • Go to Tools → Global Options
  • Under General, uncheck:
    • “Restore .RData into workspace at startup”
  • Set:
    • “Save workspace to .RData on exit” → Never

This helps avoid common beginner mistakes.

5️⃣ (Optional but Recommended) Install Required Packages

If you want to prepare a bit by installing a few resources we’ll use, in RStudio Console, copy and paste:

install.packages(c("tidyverse", "data.table", "haven", "ggplot2"))

6️⃣ If You Encounter Problems

  • Restart your computer
  • Make sure R was installed before RStudio
  • If nothing helps, write me an email: Bence.sagvari@uni-corvinus.hu
