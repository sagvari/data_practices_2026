Creating a new project

Project Setup in RStudio

Before you start analyzing data, it’s important to organize your work in an R Project. This keeps all your files, scripts, and data together in one location.

Step-by-Step Setup

  1. Open RStudio and go to File → New Project

  2. Select New Directory and choose New Project

  3. Give your project a meaningful name (e.g., week_2_analysis or penguin_study)

  4. Choose where you want to save it on your computer

  5. Click Create Project – RStudio will restart in your new project

What Just Happened?

RStudio created a folder with: - A .Rproj file (the project file) - Your working directory is now set to this folder - All your scripts and data will live here

Create Subfolders (Optional but recommended)

In the Files panel (bottom right), create these folders:

  • data/
  • scripts/
  • output/
  • misc/
